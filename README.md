## Run Locally

Clone the project

```bash
  git clone https://github.com/ViniciusCuest/SurveyForm-PB.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
Run Typescript watch mode

```bash
  tsc -w | tsc -watch
```
